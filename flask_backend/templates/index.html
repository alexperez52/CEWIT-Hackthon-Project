NBA Most Valuable Player Predictions

Sherzod Nimatullo, Alexis Perez-Mendoza, Will Krasnov, Mark Moawad


This project uses different machine learning models to predict the next NBA MVP based on previous winners and runner ups. We scraped Basketball Reference and obtained over 600 previous MVP winners and runner up. We trained our models using this dataset. Almost every feasible basketball stat was utilized and fed into a heatmap which assisted us in better understanding the corellations between win share and other stats. Using 3 different models, we created decently accurate model which does gives a decently correct prediction, with Percentage of occurance. On page Data Page & Terms, you can see the result of a linear regression plot on the data.
