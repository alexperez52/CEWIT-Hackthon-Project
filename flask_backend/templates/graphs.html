Data Techniques and Terminology

Linear Regression Model

Using Linear Regression we can determine the prediction of the dependant variable which in our case is the Award Shares. We trained this model using the data which we filtered with the Correlation Matrix and we use those inputs as the various independant variables.

Correlation Matrix

We used a Correlation Matrix to find the positive relationship between variables in order to determine which were most impactful to our model and analysis. Aside from the diagonal, Darker means stronger correlation, while lighter means weaker.
